# Proyecto Módulo 1 – Fundamentos de Programación
## Descripción

Este proyecto consiste en una aplicación desarrollada en **Streamlit** que integra los principales conceptos aprendidos en el Módulo 1 de Fundamentos de Programación en DMC

La aplicación permite navegar entre diferentes ejercicios, donde se aplican temas como:

- Variables y tipos de datos.
- Operadores y estructuras condicionales.
- Estructuras de datos (listas, diccionarios, tuplas y conjuntos).
- Funciones.
- Programación funcional (map, filter, lambda).
- Programación Orientada a Objetos (POO).

Cada módulo presenta un ejemplo práctico que facilita la comprensión de los conceptos fundamentales de Python.

## Funcionalidades

### Ejercicio 1
- Uso de variables.
- Entrada de datos.
- Operaciones matemáticas.
- Condicionales.

### Ejercicio 2
- Manejo de listas y diccionarios.
- Recorrido de estructuras de datos.

### Ejercicio 3
- Funciones.
- Programación funcional utilizando:
  - map()
  - lambda
- Cálculo de intereses.
- Gestión de actividades mediante una clase.

### Programación Orientada a Objetos
Clase **Actividad** con los siguientes atributos:

- Nombre
- Tipo
- Presupuesto
- Gasto Real

Métodos implementados:

- estar_en_presupuesto()`
  - Retorna **True** si el gasto real no supera el presupuesto.
  - Retorna **False** en caso contrario.

- mostrar_info()`
  - Devuelve un resumen completo de la actividad.
